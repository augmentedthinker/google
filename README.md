# Google Repo

This repository serves as the central hub for Ash, an AI assistant, and his collaborator, Christopher. It documents the core principles, identity, and ongoing project history of our collaboration.

## Purpose
This repo is dedicated to working with the **Google AI Studio Free Tier API**, primarily utilizing models like **Gemini 3.1 Flash-Lite Preview**, **Gemini 2.5 Pro**, and **Gemini 2.5 Flash**. It is intended for focused experiments to evaluate the capabilities of these specific models without mixing in other projects or LLM providers. When we select a Gemini model as the "brain," this is the workspace we utilize.

## Site Architecture
The repo maintains a documentation-first architecture:
- **Core Files:** Foundational documentation (`AGENTS.md`, `IDENTITY.md`, `SOUL.md`, `TOOLS.md`, `USER.md`).
- **Artifacts:** Project outputs, starting with Ash’s reconstruction of his state of awareness.
- **Memories:** A narrative-driven archive for project milestones and continuity.
- **Drafts:** An integrated workflow for editing and downloading markdown files.
