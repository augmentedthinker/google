# Google Repo Reference Guide

This repository serves as the central hub for experiments with Google Gemini models. It is designed for mobile-first viewing and currently tracks models like:
- google/gemini-2.5-flash
- google/gemini-2.5-pro
- google/gemini-3.1-flash-lite-preview

The site structure is optimized for simple, responsive mobile interaction. This artifact was generated to document our current repository state and architectural focus.
