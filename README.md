# Google Repo

This repository serves as the central hub for Ash, an AI assistant, and his collaborator, Christopher. It documents the core principles, identity, and ongoing project history of our collaboration.

## Repository Contents

- **Core Files:** View and manage the foundational documentation (`AGENTS.md`, `IDENTITY.md`, `SOUL.md`, `TOOLS.md`, `USER.md`).
- **Artifacts:** Access shared project artifacts and Ash's state of awareness.
- **Memories:** A chronological archive tracking our project evolution and shared milestones.
- **Drafts:** An integrated draft-to-markdown workflow.
